Detailed Description

Action server that allows the user to set a target point or to cancel it.

Author
Annika Delucchi
Version
1.0
Date
9/03/2024

Subscribes to:
° /odom: topic to get the actual position and velocity of the robot

Publishes to:
° /pos_vel: custom message containing the actual position and velocity of the robot

Services:
° None

Ros parameters:
° /des_pos_x: x coordinate of the last target
° /des_pos_y: y coordinate of the last target

Description:
This action server implements an action client, allowing the user to set a target point or to cancel it (this node also uses the status of the goal to know when the target has been reached); this node also publishes robot position and velocity as a custom message.

It implements different functions.

set_client_parameters() implements an action client, which also provides a user interface, running on a separate terminal, to let the user choose from the terminal either to:
° set a new target point that the robot must reach.
° cancel the goal previously chosen.

The publisher_node() function is used to create and publish a custom message containing the actual position (x,y) and velocity (linear, angular) of the robot. This function represents the callback of a subscriber, which takes the required information subscribing to the topic odom.
