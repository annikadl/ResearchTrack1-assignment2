Description\+: ~\newline
 This action server implements an action client, allowing the user to set a target point or to cancel it (this node also uses the status of the goal to know when the target has been reached); this node also publishes robot position and velocity as a custom message. ~\newline


It implements different functions. ~\newline


set\+\_\+client\+\_\+parameters() implements an action client, which also provides a user interface, running on a separate terminal, to let the user choose from the terminal either to\+: ~\newline
 ° set a new target point that the robot must reach. ~\newline
 ° cancel the goal previously chosen. ~\newline


The \mbox{\hyperlink{set__target__client_8py_ab7cf6f6222a1c32e7c2441dc9c831151}{publisher\+\_\+node()}} function is used to create and publish a custom message containing the actual position (x,y) and velocity (linear, angular) of the robot. This function represents the callback of a subscriber, which takes the required information subscribing to the topic {\ttfamily odom}.
